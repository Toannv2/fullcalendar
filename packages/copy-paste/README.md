
# FullCalendar Copy-paste Plugin

Copy and paste events using the keyboard.
### Copy:
```
[Ctrl| Cmd()] + C
```

### Paste:
```
[Ctrl | Cmd] + V
```

### Move:
```
[Ctrl | Cmd] + X
```
